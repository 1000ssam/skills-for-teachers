# handover-generator

공문서 파일명을 분석하여 **업무 인수인계서를 자동 생성**합니다.

파일 내용은 열지 않으며, **파일명만으로** 업무를 분류하고 인수인계서를 작성합니다.

---

## 설치

### 방법 1. 명령어 한 줄 (추천)

PowerShell을 열고 아래 명령어를 붙여넣은 뒤 Enter를 누르세요.

```powershell
irm https://raw.githubusercontent.com/1000ssam/skiils-for-teachers/main/skills/handover-generator/install.ps1 | iex
```

> **PowerShell 여는 방법:** `Win + R` → `powershell` 입력 → Enter

설치 후 **Claude Code를 재시작**하면 됩니다.

---

### 방법 2. 파일 직접 다운로드

1. **[메인 리포](https://github.com/1000ssam/skiils-for-teachers)** 로 이동합니다.
   > ⚠️ 이 페이지(개별 스킬 폴더)가 아닌, **메인 리포 첫 화면**으로 가야 합니다.

2. 초록색 **`<> Code`** 버튼 → **`Download ZIP`** 클릭

3. 다운로드된 ZIP 파일 압축 해제

4. 압축 해제한 폴더 안의 `skills\handover-generator` 폴더를 아래 경로에 복사합니다:
   ```
   C:\Users\{내 사용자 이름}\.claude\skills\
   ```
   > `.claude` 폴더가 안 보이면: 파일 탐색기 → **보기** → **숨긴 항목** 체크

5. Claude Code 재시작

## 사용법

Claude Code에서 이렇게 말하면 됩니다:

| 입력 | 동작 |
|------|------|
| `"인수인계서 작성해줘"` | 공문 폴더 분석 → 마크다운 인수인계서 생성 |
| `"인수인계서를 docx로 만들어줘"` | Word 문서로 추가 변환 |
| `"공문 분석해줘"` | 공문 분류 결과만 확인 |
| `"인수인계서 검증해줘"` | 기존 인수인계서와 공문 목록 교차 검증 |

---

## 출력 형식

| 형식 | 설명 |
|------|------|
| **마크다운** (.md) | 기본 출력. 노션 Import 호환 |
| **Word** (.docx) | 선택 옵션. pandoc 필요 (미설치 시 자동 설치 안내) |

## 인수인계서 구조

생성되는 인수인계서는 다음 7부 + 부록으로 구성됩니다:

| 섹션 | 내용 |
|------|------|
| **1부. 개요** | 기관 정보, 업무 영역 요약표 |
| **2부. 업무영역 상세** | 카테고리별 업무 설명, 세부 절차, 관련 공문 |
| **3부. 연간 업무 달력** | 월별(3월~2월) 주요 업무 배치 |
| **4부. 위원회 및 협의체** | 위원회 목록, 법적 근거, 구성·운영 절차 |
| **5부. 대외 발송 문서** | 가정통신문 목록, 기타 대외 문서 |
| **6부. 예산 관련 사항** | 예산 항목, 지출 절차 |
| **7부. 후임자를 위한 유의사항** | 업무 우선순위, 반복 패턴, 주의사항 |
| **부록. 공문 색인** | 전체 공문 번호순 목록 + 업무 영역 매핑 |

각 카테고리에는 `[작성자 메모]` 섹션이 포함되어, 후임자가 실무 경험을 직접 보충할 수 있습니다.
