# 📊 exam-analyzer (수능 기출 분석기)

수능/모평/학평 기출 PDF를 **분석**하거나 **문항별 이미지로 분할**하는 Claude Code 스킬입니다.

## 대응 과목

모든 수능 과목에 대응합니다:
- 사회탐구 (한국사, 동아시아사, 세계사, 생활과윤리, 사회문화 등)
- 과학탐구 (물리학, 화학, 생명과학, 지구과학)
- 국어, 수학, 영어

## 설치

PowerShell에서 실행:

```powershell
irm https://raw.githubusercontent.com/1000ssam/skills-for-teachers/main/skills/exam-analyzer/install.ps1 | iex
```

### 필요 환경
- [Claude Code](https://claude.ai/claude-code)
- Python 3.10+
- Python 패키지: `pymupdf`, `pillow`, `numpy` (설치 스크립트가 자동 설치)

## 사용법

Claude Code에서:

### 분할 모드 — 시험지를 문항별 이미지로 자동 분할

```
기출 분할 "D:\기출\2024 수능 동아시아사.pdf"
기출 분할 "D:\기출\2024\"
문항 분할 "C:\시험지\모의고사.pdf"
```

- 파일 지정 시 해당 PDF만, 폴더 지정 시 하위 PDF 전체를 처리
- 시험명 자동 식별 (PDF 텍스트 우선, 파일명 폴백)
- 입력 경로에 시험명 폴더를 생성하여 `Q01.webp`, `Q02.webp`, ... 저장

### 분석 모드 — 교과서 기반 기출 매칭 분석

```
기출 분석 03
기출 분석 미적분
기출 분석 전자기 유도 p1-3
기출 분석 조선 전기 정치
```

- 교과서 PDF가 있으면 키워드 자동 추출 (교과서 모드)
- 없으면 자연어로 주제를 지정 (자연어 모드)
- 첫 실행 시 교과서/기출 폴더 경로를 물어보고 이후 자동 기억

## 출력물

### 분할 모드

- **문항별 이미지** (`{시험명}/Q01.webp`, `Q02.webp`, ...)
  - 모든 문항을 자동 감지하여 개별 크롭
  - 과목 라벨·페이지 번호 자동 제거, 사방 균일 패딩

### 분석 모드

1. **기출 분석표** (`[단원번호]_[단원명]_기출분석.md`)
   - 시험별 출제 현황
   - 출제 패턴 분석
   - 키워드별 핵심 정리
   - 시험장 판별 체크리스트
   - 출제 경향 요약

2. **문항 스크린샷** (`[단원번호]_기출_스크린샷/`)
   - 관련 문항을 PDF에서 개별 크롭한 WebP 이미지
   - 과목 라벨·페이지 번호 자동 제거
   - 사방 균일 패딩 적용

## 기출 PDF 준비

EBSi(ebsi.co.kr) 등에서 기출 PDF를 다운로드하여 한 폴더에 모아두세요. 파일명은 상관없으며, 시험 유형은 PDF 내용에서 자동 식별합니다.

## 교과서 PDF도 준비하면 더 강력합니다

기출 분석 모드를 최대한 활용하려면, **교과서를 단원별 PDF로 분리**해서 한 폴더에 넣어두세요.

### 폴더 구조 예시

```
D:\교과서\동아시아사\
├── 01. 동아시아 역사의 시작.pdf
├── 02. 동아시아 세계의 성립과 변화.pdf
├── 03. 동아시아의 사회 변동과 문화 교류.pdf
└── ...

D:\기출\동아시아사\
├── 2026학년도 수능.pdf
├── 2025학년도 6월 모평.pdf
└── ...
```

### config 설정

첫 실행 시 경로를 물어보고 자동 저장되지만, 직접 설정할 수도 있습니다.

`~/.claude/skills/exam-analyzer/config.json`:

```json
{
  "textbookDir": "D:\\교과서\\동아시아사",
  "examDir": "D:\\기출\\동아시아사"
}
```

| 항목 | 설명 | 필수 여부 |
|------|------|-----------|
| `textbookDir` | 교과서 단원별 PDF 폴더 | 교과서 모드에서만 필수 |
| `examDir` | 기출 시험 PDF 폴더 | 항상 필수 |

### 교과서가 있으면 가능한 워크플로우

```
기출 분석 03
```
→ `03. 동아시아의 사회 변동과 문화 교류.pdf`를 자동 매칭
→ 교과서에서 키워드 자동 추출 (굵은 글씨, 고유명사, 학습 목표 등)
→ 기출 전체에서 해당 키워드가 포함된 문항을 찾아서 매칭
→ 분석표 + 관련 문항 스크린샷 생성

```
기출 분석 03 p1-3
```
→ 단원 03의 **1~3페이지만** 사용하여 분석 (소주제별로 쪼개기)

교과서 없이도 `기출 분석 조선 전기 정치`처럼 자연어로 주제를 지정하면 Claude가 키워드를 생성해서 분석해주지만, 교과서 PDF가 있으면 **실제 교과서에 나오는 용어 기준**으로 더 정밀하게 매칭됩니다.
